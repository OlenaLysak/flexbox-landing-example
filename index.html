Бесплатная версия Visual Studio
Visual Studio Mobile Center,
предварительная версия
Visual Studio Mobile Center — это центр управления вашими мобильными приложениями.
Приложение, написанное на любом языке, можно перенести в облачные службы и службы жизненного цикла Visual Studio Mobile Center, чтобы ускорить выпуск, повысить качество приложений, а также получить время и возможности для удовлетворения потребностей пользователей.

Все непрерывно
Автоматизируйте весь жизненный цикл приложения, выполнив несколько простых действий.
Войдите в систему и подключите свои приложения Android, iOS и Windows к функциям непрерывной интеграции, автоматического тестирования, распространения, мониторинга и взаимодействия в Visual Studio Mobile Center. Используйте рабочие процессы, сокращайте циклы выпуска и высвобождайте время для работы над функциями.
Создание высококлассных приложений
Выпускайте высококачественные приложения и повышайте лояльность пользователей.
Вы можете внедрять автоматические тесты пользовательского интерфейса в процесс сборки и распространения, благодаря чему ваше приложение тестируется на сотнях устройств еще до того, как будет доставлено пользователю. Создание отчетов о сбоях и богатые аналитические возможности позволяют оперативно извлекать диагностические и ценные сведения на основе сообщений и действий пользователей в рабочей среде.
Гибкость и широкий выбор
Повысьте скорость разработки и качество своих продуктов с помощью уже имеющихся у вас инструментов.
Оптимизированный рабочий процесс Mobile Center подойдет для любого выбранного языка программирования — Objective-C, Swift, Java, Xamarin или React Native. У вас есть собственное решение CI? Используете другую платформу для распространения бета-версий? Это не проблема. Наши открытая платформа и API позволяют интегрировать лишь те компоненты Mobile Center, которые вам требуются.
Наша стратегия развития
Мы с нетерпением ждем возможности продемонстрировать весь потенциал этой системы.
Visual Studio Mobile Center является следующим поколением таких решений, как HockeyApp и Xamarin Test Cloud. Сейчас поддерживается следующее:
Мобильные платформы Objective-C, Swift, Java, Xamarin, React Native, UWP
Функции жизненного цикла Сборка, тестирование, распространение, аналитика, сбои
Облачные функции Проверка подлинности, хранилище данных, Push-уведомления, автономная синхронизация
И это только начало! У нас есть грандиозные планы относительно того, как уже в ближайшем будущем помочь вам быстрее выпускать более качественные мобильные приложения с поддержкой большего числа функций жизненного цикла и мобильных облачных служб.
